# What is Nebula?

Nebula is an open-source **Backend-as-a-Service (BaaS)** built with Go that provides essential backend infrastructure for your applications. It allows you to quickly bootstrap applications with authentication, dynamic database management, and RESTful APIs.

## Key Features

<AccordionGroup>
  <Accordion title="Dual Authentication" icon="shield-halved">
    - **JWT Tokens** for user management and account operations
    - **API Keys** for database-scoped programmatic access
    - Secure password hashing with bcrypt
  </Accordion>
  
  <Accordion title="Dynamic Database Management" icon="database">
    - Create isolated SQLite databases per user
    - Define custom table schemas on-the-fly
    - Full CRUD operations on records
  </Accordion>
  
  <Accordion title="Built-in Security" icon="lock">
    - IP-based rate limiting
    - CORS support with configurable origins
    - Centralized error handling
  </Accordion>
  
  <Accordion title="Container Ready" icon="docker">
    - Docker and Docker Compose support
    - Kubernetes deployment manifests
    - Persistent volume support for data
  </Accordion>
</AccordionGroup>
